# Full Stack Java Project

## Overview
This is a comprehensive full-stack Java project that utilizes Java, JavaScript, HTML, and CSS to build a robust web application.

## Tech Stack
- **Java**: Main programming language for backend development.
- **JavaScript**: For frontend development and client-side interactions.
- **HTML**: Markup language for structuring web pages.
- **CSS**: Styles for the web pages, enhancing the user interface.

## Features
- User authentication and authorization.
- CRUD (Create, Read, Update, Delete) operations on server-side.
- Responsive design for various screen sizes.
- RESTful API for connecting frontend and backend.

## Installation
1. **Clone the repository**:
   ```bash
   git clone https://github.com/KhandelwalTapan7/full-stack-java-project.git
   ```

2. **Navigate to the project directory**:
   ```bash
   cd full-stack-java-project
   ```

3. **Build the project**:
   ```bash
   ./mvnw clean install
   ```

4. **Run the application**:
   ```bash
   ./mvnw spring-boot:run
   ```

## Usage
- Open a web browser and navigate to `http://localhost:8080` to access the application.

## Testing
- To run tests, use the command:
  ```bash
  ./mvnw test
  ```
